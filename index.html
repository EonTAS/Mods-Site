Welcome

This website is here to show how my project to completely rewrite Project M's debug mode in a way that utilises the graphics engine to draw straight to the screen instead of uses individual graphics objects in the games limited gfx system, as well as add new features to it such as background toggling and accurate hitbox interpolation.
The general idea of this is to mimmick Smash Bros Melee's debug mode and allow it to be a much more complete feature.
