Collision Display:

Allows users to view how the game interprets hitboxes and hurtboxes to better understand whether something should hit and how interactions play out
Also allows completely disabling the fighters render so that its easier to see the collisions.

completion
Status Module Display:

Prints text to the screen detailing the exact action each fighter is in, as well as details on their subaction and what frame of animation they are on.

completion
Stage Collisions Display:

Allows you to disable the stage render entirely and instead draw the exact stage collisions the game uses.
This also allows viewing blastzones and camera boundaries.
